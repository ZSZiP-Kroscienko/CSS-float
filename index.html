5. Problem: Zawalenie Kontenera

Gdy elementy mają float, kontener traci wysokość! Rozwiązanie: overflow: hidden;

BEZ overflow: hidden (zawalony kontener):

Item 1
Item 2
Item 3

Kontener ma wysokość = 0! (tekst wylatuje poza)

Z overflow: hidden (naprawiony):

Item 1
Item 2
Item 3

Kontener ma poprawną wysokość!

.container { overflow: hidden; /* ← Naprawia zawalenie */ }

8. Clear Property - Czyszczenie Float

clear zatrzymuje opływanie tekstu.

Floated Box

Czyszczenie: clear: left; - Tekst już nie opływa!
.clear { clear: left; /* Czyszczenie float left */ }
.clear-both { clear: both; /* Czyszczenie obu */ }

Float to starszy sposób na opływanie. Dzisiaj lepiej użyć Flexbox lub Grid!
