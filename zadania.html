Float - Zadania Praktyczne dla Uczniów

Zadanie 1: Obraz z tekstem (float: left)

Cel: Umieść obraz z lewej strony, tekst opływa z prawej.
Co użyć: float: left; + margin-right

Twój rezultat (podgląd):

Zdjęcie

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.

CSS do uzupełnienia:

.task-1-image {
  /* DODAJ: float: left; */
  /* DODAJ: margin-right: 15px; */
  width: 150px;
  height: 120px;
  background-color: #007bff;
}

Zadanie 3: Dwukolumnowy layout

Cel: Sidebar (25%) z lewej, zawartość (75%) z prawej za pomocą float.
Co użyć: float: left; + width + overflow: hidden;

Twój rezultat (podgląd):

Sidebar

  • Menu 1
  • Menu 2
  • Menu 3

Główna zawartość

Lorem ipsum dolor sit amet, consectetur adipiscing elit.

CSS do uzupełnienia:

.task-3 {
  /* DODAJ: overflow: hidden; */
  background-color: #f0f0f0;
}

.task-3-sidebar {
  /* DODAJ: float: left; */
  /* DODAJ: width: 25%; */
  background-color: #e8f4f8;
  padding: 15px;
}

.task-3-main {
  /* DODAJ: float: left; */
  /* DODAJ: width: 75%; */
  padding: 15px;
}

Zadanie 5: Clear - Zatrzymanie opływania

Cel: Zatrzymaj tekst przed opływaniem unoszącego się elementu.
Co użyć: clear: left; lub clear: both;

Twój rezultat (podgląd):

Unoszący się element

Tekst opływa element...

Tekst opływa element...

To już nie opływa!

CSS do uzupełnienia:

.task-5-floated {
  /* DODAJ: float: left; */
  width: 150px;
  background-color: #007bff;
  color: white;
  padding: 10px;
  margin-right: 10px;
  margin-bottom: 10px;
}

.task-5-cleared {
  /* DODAJ: clear: left; */
  /* lub: clear: both; */
  background-color: #ffc107;
  padding: 10px;
  margin-top: 10px;
}

Zadanie 6: Float: none (Brak opływania)

Cel: Zmień float: none na elementy, które powinny być normalne.
Co użyć: float: none;

Twój rezultat (podgląd):

Box 1 (float: left)
Box 2 (float: none)
Box 3 (float: none)

CSS do uzupełnienia:

.task-6-box {
  background-color: #007bff;
  color: white;
  padding: 20px;
  margin: 5px;
}

.task-6-float {
  /* DODAJ: float: left; */
  width: 30%;
}

.task-6-none {
  /* DODAJ: float: none; */
}

Zadanie 7: Float: inherit (Dziedziczenie)

Cel: Element dziedziczy float od rodzica.
Co użyć: float: inherit;

Twój rezultat (podgląd):

Dziedziczy float

Tekst opływa dziecko (bo dziedziczy float z rodzica)

CSS do uzupełnienia:

.task-7-parent {
  /* DODAJ: float: left; */
  width: 40%;
  background-color: #e8f4f8;
  padding: 15px;
  border: 2px solid #007bff;
}

.task-7-child {
  /* DODAJ: float: inherit; */
  width: 50%;
  background-color: #007bff;
  color: white;
  padding: 10px;
}
